Iwanan ni Thomas Dohmke, CEO ng GitHub, ang kanyang posisyon upang magtatag ng bagong startup.

Inihayag ni Thomas Dohmke, ang Chief Executive Officer ng GitHub, na siya ay magbibitiw sa kanyang posisyon sa katapusan ng 2025 upang magtatag ng bagong startup, matapos mamuno sa Microsoft-owned code-hosting platform sa loob ng halos apat na taon. Si Dohmke ay naging CEO noong huli ng 2021 at sinabi niyang mananatili siya hanggang 2025 upang suportahan ang nakaplano na paglipat ng pamunuan.

Sa kanyang panunungkulan, pinalakas ng GitHub ang kanilang pokus sa artificial intelligence at inilunsad ang GitHub Copilot, isang AI coding assistant na sinasabing ginagamit ng humigit-kumulang 20 milyong developer. Mula nang mag-umpisa si Dohmke, nakaranas ang kumpanya ng mabilis na pag-unlad at makabuluhang inobasyon, na ang Copilot ang pinakamatingkad na produktong konektado sa pagbabagong ito.

Sa ilalim ng pamumuno ni Dohmke, lumawak ang GitHub, na nagho-host ng 1 bilyong repositories at forks at nagsisilbing komunidad para sa mahigit 150 milyong developer. Lumipat ang kumpanya sa isang remote-first engineering model na binanggit ni Dohmke bilang dahilan ng kanyang kasiyahan sa kanyang anunsyo. Itinaguyod niya ang kanyang pag-alis bilang pagbabalik sa kanyang ugatang negosyante, isinulat, "Matapos ang lahat ng panahong ito, muling nananabik ang aking ugat ng startup at nagpasiya akong iwanan ang GitHub upang muling maging isang tagapagtatag."

Inihayag ng Microsoft na hindi sila maghahahanap ng bagong CEO para sa GitHub pagkatapos ng pag-alis ni Dohmke. Sa halip, ang pamunuan ng GitHub ay direktang mag-uulat sa ilang mga ehekutibo ng Microsoft at mas mahigpit na magiging integradong bahagi ng CoreAI division ng Microsoft. Ang pagbabago sa istruktura na ito ay tinukoy ng Microsoft bilang hakbang para mas lalo pang yakapin ang AI sa kanilang ecosystem at patuloy na suportahan ang komunidad ng mga developer.

Ang muling pagbuo ng organisasyon ay umaayon sa mas malawak na mga inisyatiba ng AI ng Microsoft. Sa pamamagitan ng pagtatanggal ng posisyon ng nakatayong CEO at pag-uulat sa pamamagitan ng maraming ehekutibo ng Microsoft at CoreAI, mas magiging mas malapit ang koordinasyon sa mga pangunahing programa ng AI ng Microsoft. Ang estrukturang ito ay nagbabawas ng autonomiya ng GitHub habang mas pinagsasama ito sa agenda ng AI ng Microsoft.

Isinasaalang-alang din ng GitHub ang kompetisyon sa AI-assisted development sa kanilang kamakailang estratehiya sa produkto. Pinalawak ng kumpanya ang Copilot at mga kaugnay na proyekto sa gitna ng tumataas na kumpetisyon mula sa mga kumpanya tulad ng Google at Cursor, na nagpapakita ng isang merkado na pabor sa mabilis na pag-develop ng mga tampok at mas malawak na integrasyon ng platform.

Ipinahayag din ni Dohmke ang panahon ng AI bilang isang malaking pagkakataon para sa paglikha ng software, hinuhulaan ang isang "bago at gintong pagkakataon ng software" at inaasahang nakabitin ang bilyong AI agents upang mas maraming developer ang makalahok. Hindi niya ibinunyag ang detalye tungkol sa kanyang planong startup, kaya't limitado ang kaalaman ng publiko sa kanyang susunod na hakbang o kung paano maaaring maapektuhan ng kanyang pag-alis ang mga prayoridad sa produkto ng GitHub.

Para sa mga developer, negosyo, at kasosyo, ang agarang implikasyon ay may dalawang bahagi: mananatili ang GitHub sa ilalim ng pag-aari ng Microsoft ngunit may pinabuting pamamahala, at maaaring magbago ang mga roadmap ng produkto upang ipakita ang mas malalim na integrasyon sa mga prayoridad ng CoreAI ng Microsoft. Ang pagbabagong ito ay maaaring mapabilis ang pagpapalabas ng mga tampok na pinapagana ng AI ngunit nagdadala rin ng mga tanong kung paano haharapin ang mga independiyenteng desisyon sa produkto.

Ang panahon ng paglipat hanggang sa katapusan ng 2025 ay nagbibigay sa Microsoft ng oras upang ipatupad ang bagong estruktura ng pag-uulat at para sa GitHub na maayos na makipag-ugnayan sa CoreAI. Dapat bantayan ng mga tagamasid ang anunsyo na nagdedetalye ng eksaktong mekanika ng pangangasiwa, ang alokasyon ng mga yaman sa engineering sa mga inisyatiba ng AI, at anumang pampublikong pahayag mula kay Dohmke tungkol sa kanyang startup upang sukatin ang mga pangmatagalang epekto sa GitHub at ang ecosystem ng mga developer.
